# LogGrep

Простой CLI-инструмент для поиска строк в файлах и каталогах (минимальный аналог grep).

## Возможности
- Поиск в одном файле или рекурсивно по каталогу
- Выводит путь файла и номер строки с совпадением (`<path>:<line>:<text>`)
- Флаг `-i` для нечувствительного к регистру поиска
- Тестовые входные данные и примеры в каталоге `demo/`

## Требования
- Go 1.24+

## Сборка и запуск
```bash
# собрать бинарник
make build        # результат: ./main

# запустить без сборки
go run main.go [флаги] <path> <needle>
```

## Использование
```bash
./main [флаги] <path> <needle>
```
- `path` — путь к файлу или каталогу; для каталога обход рекурсивный
- `needle` — искомая подстрока
- Флаги:
  - `-i` — игнорировать регистр

### Примеры
```bash
# найти ERROR в конкретном файле
go run main.go demo/app.log ERROR

# нечувствительный поиск "error" по всем файлам каталога
go run main.go -i demo error
```
